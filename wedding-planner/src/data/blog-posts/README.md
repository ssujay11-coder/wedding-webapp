## Usage Instructions

### For Development
1. These JSON files can be imported into your blog system
2. Each file contains all metadata and content in structured format
3. Content field contains markdown-formatted text ready for rendering

### For CMS Integration
```javascript
// Example: Reading and parsing blog posts
import blogPosts from './data/blog-posts/*.json';

// Access metadata
const title = blogPost.title;
const slug = blogPost.slug;
const excerpt = blogPost.excerpt;

// Render content (markdown to HTML)
const htmlContent = markdownToHtml(blogPost.content);
```

### For SEO
- Use `keywords` array for meta keywords
- Use `excerpt` for meta description
- Use `title` for page title and H1
- Use `slug` for URL structure (example.com/blog/{slug})
